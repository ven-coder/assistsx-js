# 更新日志

## 0.2.1（2026-04-22）

- 步骤器默认最大重复次数 15，修正无限重复判断。

## 0.2.0（2026-04-09）

- 无障碍事件过滤、应用信息管理相关能力。
- README 等文档更新。

## 0.1.42（2026-03-18）

- 构建迁移至 tsup，输出 ESM / CommonJS / 全局包；`main` 指向 CJS，`unpkg` / `jsdelivr` 指向全局构建。
- `package.json` 增加 `./global` 导出；README 补充通过 CDN 的 HTML 引用示例。

## 0.1.41（2026-03-18）

- 屏幕尺寸接口与数据规范化；节点边界以 `bounds` 为主（`boundsInScreen` 弃用提示）。

## 0.1.36（2026-02-12）

- WebView 悬浮窗位置与样式等配置项；关闭悬浮窗（`closeOverlay`）。
- 状态栏 / 导航栏工具、悬浮窗基础 API。

## 0.1.29（2026-02-06）

- 剪贴板文本读取；ML Kit 屏幕文字识别及短语位置相关 API。

## 0.1.27（2026-02-01）

- HTTP 下载支持写入系统相册；相册模块声明与导出；步骤完成回调逻辑修复。

## 0.1.25（2026-01-26）

- 截图保存到文件（含节点截图）；图片处理工具（缩放、裁剪、旋转等）；下载与相册能力衔接。

## 0.1.21（2026-01-08）

- 版本线进入 0.1.x：屏幕尺寸、`takeScreenshotSave`、根节点树 JSON 导出；步骤异常重试；文件写入与 HTTP 多文件上传等增强。

## 0.0.3004 前后（2025-12 至 2026-01）

- 输入法（IME）相关 API。
- 调用返回值与 Bounds 等类型校验、默认值与步骤控制等小版本迭代。

## 0.1.x 之前（0.0.13xx–0.0.20xx 阶段）

- **自动化与步骤**：`AssistsX` / `Node` 类化、同步与异步调用、步骤器与步骤状态存储、拦截器与备用步骤、手势与截图、节点查找与可见性 / 滚动。
- **系统与 Web**：无障碍事件监听与过滤、WebView 悬浮窗与扫码、系统浏览器、屏幕常亮、长按自动粘贴、设备标识（如 Android ID）。
- **网络与媒体**：文件读写、HTTP 下载与上传、相册与音频、联系人、截图文字识别等。
- **节点与稳定性**：丰富无障碍属性、屏幕坐标边界、按标签查找父节点、焦点与异步节点超时；修复 JSON 中含控制字符时的解析问题。
- **工程与协议**：MIT 许可证；包名与仓库定位为 assistsx-js；TypeScript 与全局类型定义。

## 项目初期
